Exemplos de DOM e Eventos

DOM (Document Object Model): modelo de objetos do documento, ou seja, a estrutura da página HTML representada como objetos que o JS pode acessar e manipular.

Para acessar elementos do DOM, o JS utiliza funções/métodos especiais de seleção. E a partir da seleção do elemento, o JS permite (através de outras funções e propriedades) realizar manipulações de diversas formas. Exemplos:

  • Menu responsivo/hamburguer
  • Modo Escuro (Dark/Light)
  • Validações personalizadas/avançadas de formulário
  • Janela Modal (caixa de diálogo)
  • Galerias de imagem, sliders/carrosséis de conteúdo
